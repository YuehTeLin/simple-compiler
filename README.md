## Built With

* [PEG.js] - A pure JavaScirpt parser generator.

## Getting Started

Fork or download the content of the repository to your computer.

### Prerequisites

- Install [Node.js] which includes [Node Package Manager][npm]

### Installing

- Install the dependencies:

```
npm install
```

And run

```
node compiler.js
```
